\# E-Commerce Data Pipeline



End-to-end ETL pipeline for e-commerce analytics using Python, PostgreSQL, Airflow, and BI tools.



\## Project Structure

\- data/ – Raw, staging, and processed data

\- scripts/ – Data generation, ingestion, transformation, quality checks

\- sql/ – DDL, DML, analytical queries

\- dashboards/ – Tableau/Power BI dashboards

\- docker/ – Containerization setup

\- tests/ – Unit and integration tests



\## Tech Stack

Python, Pandas, PostgreSQL, Docker, Airflow, Tableau/Power BI
